# 註冊登入系統

- 此作品是使用 Springboot 搭配 Hibernate 以及 mySQL 來模擬登入註冊的作品
- 另外還有搭配使用 javaMail 來做忘記密碼的信件寄送，以及密碼使用 md5 做加密
